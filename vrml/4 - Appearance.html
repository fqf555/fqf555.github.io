4 - Appearance

Material

Material {
diffuseColor 0.8 0.8 0.8 # 漫反射颜色（最主要）
transparency 0 # 透明度
specularColor 0 0 0 # 镜面反射光
emissive 0 0 0 # 自发光颜色
ambientIntensity 0.2 # 有多少环境光被物体表面反射（一般不改）
shiniess 0.2 # 物体表面的亮度（一般不改）
}

texture

纹理是在二维平面内的，x 轴和 y 轴分别对应 s 轴和 t 轴，因此下面属性中的向量也是二维的。

纹理在所有面都会粘。
ImageTexture {
url "" # 也可以用 [] 指定多个备选素材，优先级依次降低
repeatS TRUE
repeatT TRUE
}
MovieTexture {
url ""
loop FALSE
speed 1.0
startTime 0.0 # 默认：不设置（一般接收发来的信息）
stopTime 0.0 # 默认：不设置（一般接收发来的信息）
repeatS TRUE
repeatT TRUE
duration_changed # 播放时长（出事件）（未加载：-1）
isActive # 正在播放？（出事件：TRUE/FALSE）
}
TextureTransform {
scale 1.0 1.0
rotation 0.0
translation 0.0 0.0
center 0.0 0.0
}
