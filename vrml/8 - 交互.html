8 - 交互

用于交互的七种传感器

这七种传感器按照感知方式分为两类（每一大类中的第一个是重点）：
  • 鼠标操作：TouchSensor, PlaneSensor, CylinderSensor, SphereSensor
  • 视点位置：ProximitySensor, VisibilitySensor, Collision
TouchSensor 和 PlaneSensor 是重点

TouchSensor

TouchSensor {
enabled TRUE
isOver # 悬停 TRUE，离开 FALSE
isActive # 按下 TRUE，抬起 FALSE
touchTime # 【重要】isOver && isActive 时输出，鼠标单击就输出
hitPoint_changed
hitNormal_changed
hitTexCoord_changed
}
touchTime：当 isOver=TRUE, isActive=TRUE时，引发该事件，所发送的值是当前时间。经常用此事件来控制当用户点击鼠标左键放开后触发的动作。

【例】利用 TouchSensor 触摸传感器节点，对圆锥位移动画进行控制。当光标移动到圆锥造型上时，会出现一个小手图标，表示这是一个被感应被控制的造型。此时，按下鼠标左键不放，圆锥动画开始，直到鼠标松开为止。
# VRML V2.0 utf8
DEF tt Transform {
children [
Shape {
geometry Box {
size 1 1 1
}
appearance Appearance {
material Material {
diffuseColor 1 1 0
}
}
}
]
}
DEF time TimeSensor {
# 这里不能写 loop TRUE，因为 loop 现在被 TouchSensor 控制
cycleInterval 3
}
DEF oo OrientationInterpolator {
key [0 0.5 1]
keyValue [
0 1 0 0
0 1 0 3.14 # 转半圈
0 1 0 6.28 # 转一圈
]
}
ROUTE time.fraction_changed TO oo.set_fraction
ROUTE oo.value_changed TO tt.rotation
# 一般 TouchSensor 就这么写，里面什么也不写
DEF touch TouchSensor {}
# 1题
ROUTE touch.isActive TO time.loop
# 2题
ROUTE touch.touchTime TO time.startTime
注意：鼠标松开之后，动画不会立即停止，而是会播放完当前周期后才停止。

PlaneSensor 平面传感器

PlaneSensor {
enabled TRUE
offset 0 0 0
autoOffset TRUE
minPosition 0 0
maxPosition -1 -1
isActive
trackPoint_changed
translation_changed
}
#VRML V2.0 utf8
DEF tt Transform {
rotation 1 0 0 3.14
children [
Shape {
geometry Cone {}
appearance Appearance {
material Material {
diffuseColor 1 0 0
}
}
}
]
}
# - 1 - 不限制移动范围
DEF pl PlaneSensor {}
ROUTE pl.trackPoint_changed TO tt.translation # 两个目前无区别
ROUTE pl.translation_changed TO tt.translation
# - 2 - 限制移动范围
DEF pl PlaneSensor {
offset 2 2 0 # 点击鼠标后先立刻跳到 2 2 0
maxPosition 3 4 # 限制区域
}
ROUTE pl.trackPoint_changed TO tt.translation # 不受限
ROUTE pl.translation_changed TO tt.translation # 受限

CylinderSensor 圆柱体传感器

CylinderSensor {
enabled TRUE
offset 0
autoOffset TRUE
minAngle 0
maxAngle -1
diskAngle 0.262 # 不用
isActive
trackPoint_changed

}

SphereSensor {

ProximitySensor 接近传感器

ProximitySensor {
enabled TRUE
center 0 0 0
size 0 0 0
isActive
position_changed
orientation_changed
enterTime
exitTime
}

VisibilitySensor 可视传感器

VisibilitySensor {
enabled TRUE
center 0 0 0
size 0 0 0
isActive
enterTime
exitTime
}

Collision 碰撞传感器

Collision {
children []
collide TRUE
proxy NULL
bboxCenter 0 0 0
bboxSize -1 -1 -1
collideTime
addChildren
removeChildren
}
